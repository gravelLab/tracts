tracts.phase_type.base_phase_type.PhaseTypeDistribution#

class PhaseTypeDistribution#

Bases: ABC

A class representing the phase-type distribution of tract lengths generated by a given (pair of) migration matrix (matrices).

maxLen#

The maximum tract length considered in the distribution. Used to speed up computations by ignoring tracts longer than maxLen, which are very unlikely to be observed in real data. If None, considers tracts of all lengths.

Type:

float

num_populations#

The number of populations in the model, inferred from the migration matrix.

Type:

int

migration_matrix#

The migration matrix defining the demographic model. Used only under monoecious admixture.

Type:

npt.ArrayLike

migration_matrix_f#

The migration matrix describing migration of female individuals, used only under dioecious admixture.

Type:

npt.ArrayLike

migration_matrix_m#

The migration matrix describing migration of male individuals, used only under dioecious admixture.

Type:

npt.ArrayLike

t0_proportions#

The proportions of tracts that originate from each population at time t=0. Computed using Eq. (F-27) in the manuscript.

Type:

npt.ArrayLike

t0_proportions_f#

The proportions of maternally-inherited tracts that originate from each population at time t=0. For autosomes, computed using Eq. (F-27) in the manuscript. For the X chromosome, computing using the recursive equations (F-28) and (F-29) in the manuscript.

Type:

npt.ArrayLike

t0_proportions_m#

The proportions of paternally-inherited tracts that originate from each population at time t=0. For autosomes, computed using Eq. (F-27) in the manuscript. For the X chromosome, computing using the recursive equations (F-28) and (F-29) in the manuscript.

Type:

npt.ArrayLike

transition_matrices#

A list of the transition matrices for tracts originating from each population, used only under monoecious admixture.

Type:

list of npt.ArrayLike

transition_matrices_f#

A list of the transition matrices for maternally-inherited tracts, used only under dioecious admixture.

Type:

list of npt.ArrayLike

transition_matrices_m#

A list of the transition matrices for paternally-inherited tracts, used only under dioecious admixture.

Type:

list of npt.ArrayLike

S0_list#

A list containing the sum across columns of every transition matrix in transition_matrices.

Type:

list of npt.ArrayLike

S0_list_f#

A list containing the sum across columns of every transition matrix in transition_matrices_f.

Type:

list of npt.ArrayLike

S0_list_m#

A list containing the sum across columns of every transition matrix in transition_matrices_m.

Type:

list of npt.ArrayLike

inverse_S0_list#

A list containing the sum across columns of the inverse of every transition matrix in transition_matrices, used only under monoecious admixture.

Type:

list of npt.ArrayLike

inverse_S0_list_f#

A list containing the sum across columns of the inverse of every transition matrix in transition_matrices_f, used only under dioecious admixture.

Type:

list of npt.ArrayLike

inverse_S0_list_m#

A list containing the sum across columns of the inverse of every transition matrix in transition_matrices_m, used only under dioecious admixture.

Type:

list of npt.ArrayLike

alpha_list#

A list of the initial probability vectors for tracts originating from each population, used only under monoecious admixture.

Type:

list of npt.ArrayLike

alpha_list_f#

A list of the initial probability vectors for maternally-inherited tracts, used only under dioecious admixture.

Type:

list of npt.ArrayLike

alpha_list_m#

A list of the initial probability vectors for paternally-inherited tracts, used only under dioecious admixture.

Type:

list of npt.ArrayLike

loglik(bins, Ls, data, num_samples, cutoff=0)#

Calculates the log-likelihood in a Poisson Random Field. This function is used to fit model parameters.

Parameters:

  • bins (ndarray) – A point grid on \((0, L)\) where the histogram values have to be evaluated.

  • Ls (ndarray) – A vector of chromosome lengths for every chromosome considered in the data.

  • data (list[list[int]]) – A list of length equal to the number of populations, where the entry at index \(i\) is a list of the observed tract counts in every bin for population \(i\).

  • num_samples (int) – The number of haploid genomes in the sample. Used to scale the predicted tract counts, as the Phase-type distribution is defined for a single haploid genome. If num_samples is zero, the log-likelihood is set to zero, as the data does not contain any information.

  • cutoff (int) – The index of the first bin to consider in the likelihood calculation. Used to ignore very short tracts, which are very likely to be false positives in real data.

Returns:

The log-likelihood of the data given the model parameters, assuming a Poisson Random Field model.

Return type:

float

Populate the density values for the phase-type distribution on the given bins, under the finite chromosome model.

  • population_number (int) – The population of interest whose tract length distribution has to be computed.

  • ETL (float) – The expected tract length under the finite chromosome model.

  • prop_connected (float) – The probability of a tract to being drawn from the Phase-type process with connected state space.

  • prop_isolated (float) – The probability of a tract being drawn from the Phase-type process with an isolated single state.

  • exp_Sx_per_bin (Union[Buffer, _SupportsArray[dtype[Any]], _NestedSequence[_SupportsArray[dtype[Any]]], bool, int, float, complex, str, bytes, _NestedSequence[bool | int | float | complex | str | bytes]]) – A list of the precomputed values of \(e^{\mathbf{S}x}\) for every bin, where \(\mathbf{S}\) is the transition matrix for the Phase-type process.

  • s1 (float | None) – The sex of the individual at generation 1. If s1 = 0 (resp. 1), only alleles paternally (resp. maternally) inherited alleles are considered. If set to None, tracts on both copies are combined.

  • alpha (Union[Buffer, _SupportsArray[dtype[Any]], _NestedSequence[_SupportsArray[dtype[Any]]], bool, int, float, complex, str, bytes, _NestedSequence[bool | int | float | complex | str | bytes]]) – The initial probability vector for the Phase-type process.

Returns:

  • npt.ArrayLike – The sorted and truncated bins on \((0, L)\) where the density will be evaluated.

Computes the tract length histogram or density on a point grid using a PhaseTypeDistribution object. This function considers an infinite chromosome.

  • density (bool) – If True, computes the Phase-type density. Else, returns the histogram values on the grid.

If density is True, the density evaluated on bins. Otherwise, the histogram values on every interval defined by bins.
